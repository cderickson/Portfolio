Saving and Exporting

Back to Project Description.

Save/Delete Session Data

File → Save Data
Data → Delete Saved Session

All tabular data and setting information is saved to the /save/ folder.

Previously Imported Files

All previously imported GameLog and DraftLog files are copied and saved to the /gamelogs/ folder and /draftlogs/ folder, respectively.

Exporting Tables

File → Save Data

MTGO-Tracker creates (5) tables: Matches, Games, Plays, Drafts, Draft Picks. Any or all of these tables can be saved as either .csv or .xlsx files. Filtered tables can also be exported.

Exported files are saved to the /export/ folder by default.
