Settings

Import Folders

Data → Set Default Import Folders

MTGO default GameLog location:
        C:\Users\[User]\AppData\Local\Apps\2.0

MTGO default DraftLog location:
        C:\Users\[User]\Documents

Select the folders containing the GameLog and DraftLog files saved by MTGO.

Export Folder

Export → Set Default Export Folder

Exported files are saved to the /export/ folder by default.

Main Window Size

  • Small: 1000x500px
  • Large: 1750x750px

Hero

Data → Set Default Hero

Set the default Hero to your MTGO username. This will move your username into the P1 column by default, making the tables more readable. Applying this setting will also activate the Statistics window, allowing you to view dashboards with data shown from the Hero's perspective.
